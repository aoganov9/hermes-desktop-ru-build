# Hermes Desktop RU Build

Russian-localized Hermes Desktop build for macOS Apple Silicon (`arm64`).

Русифицированная сборка Hermes Desktop для macOS Apple Silicon (`arm64`).

This repository is for people searching for:

- Hermes Desktop Russian version
- Hermes Desktop на русском
- Hermes Desktop RU localization
- Hermes Desktop translation
- Hermes macOS Russian build
- Hermes Desktop Apple Silicon build

## What This Is

This is an unofficial community build of Hermes Desktop with Russian interface translation.

## Что это

Это неофициальная community-сборка Hermes Desktop с русским интерфейсом.

Target platform:

- macOS
- Apple Silicon (`arm64`)

Целевая платформа:

- macOS
- Apple Silicon (`arm64`)

## Important

- This is not an official Hermes release.
- The app build is unsigned.
- macOS may ask you to allow the app manually in `Privacy & Security`.

## Важно

- Это не официальный релиз Hermes.
- Сборка не подписана Apple Developer ID.
- macOS может попросить вручную разрешить запуск через `Privacy & Security`.

## Download

Download the latest build from the GitHub `Releases` section of this repository.

## Скачать

Скачать актуальную сборку можно в разделе `Releases` этого репозитория.

После скачивания:

1. распакуйте архив;
2. перенесите `Hermes.app` в `Applications` или запустите напрямую;
3. если macOS заблокирует запуск, разрешите его в `Privacy & Security`.
